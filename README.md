## Dependencies

  ### • [SFML](https://www.sfml-dev.org/download.php) - Simple and Fast Multimedia Library
  ### • [CMake](https://cmake.org/download/) - Build Generator System

## Generate build files
  
    cmake -S . -B build

## Install
    
    cmake --build build --config Release --target install
